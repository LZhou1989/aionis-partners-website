CSS Loading Test

If you see this box in blue with white text, the CSS file is loading correctly.

This means the styles.css file is being found and applied.

If you see this box in red, the CSS file is NOT loading.

This indicates a problem with the file path or server setup.

Troubleshooting Steps:

  1. Check File Structure: Make sure styles.css is in the same folder as index.html
  2. Use a Local Server: Don't open the file directly - use a local server instead
  3. Check Browser Console: Press F12 and look for errors in the Console tab
  4. Try Different Browser: Test in Chrome, Firefox, or Edge

How to Set Up a Local Server:

Option 1 - Python:

python -m http.server 8000

Then open: http://localhost:8000

Option 2 - Node.js:

npx http-server

Option 3 - VS Code:

Install "Live Server" extension and right-click on index.html → "Open with Live Server"
